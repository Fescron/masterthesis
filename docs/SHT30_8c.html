◆ softResetSHT30()

int8_t softResetSHT30 ( void  )

Soft reset the SHT30.

This method also initializes i2C if necessary.

Note
This is a static method because it's only internally used in this file and called by other methods if necessary.
Returns
  • 0 - Byte written correctly.
  • -1 - i2cTransferNack, NACK received during transfer.
  • -2 - i2cTransferBusErr, Bus error during transfer (misplaced START/STOP).
  • -3 - i2cTransferArbLost, Arbitration lost during transfer.
  • -4 - i2cTransferUsageFault, Usage fault.
  • -5 - i2cTransferSwFault, SW fault.
  • -6 - Coudldn't configure I2C.
