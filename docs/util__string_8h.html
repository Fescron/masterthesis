Convert an ASCII char array (string) to a hexadecimal char array.

Note
If the input is not a string (ex.: "Hello world!") but a char array, the input message (array) needs to end with NULL ("\0")!
Parameters
[in]inbufThe given ASCII string to convert to a hexadecimal string.
[out]outbufThe buffer to put the resulting string in.
This needs to have twice the length of the input message + 1! (two HEX characters for one ASCII character + NULL termination)
